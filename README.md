# 5歳児健診でつなぐ こどもまんなか社会

日本小児保健協会 健やか親子21・成育医療等基本方針推進委員会の提言「5歳児健康診査を活用した『こどもまんなか社会』の実現に向けた啓発の推進」（2026年5月21日）に基づく、**保護者向け啓発資材**のプレビュー。

## 位置づけ

- **委員会内プレビュー**（限定共有）
- 検索エンジンには掲載されない設定（noindex、robots.txt）
- 後続成果物（ポスター、リーフレット）の内容ソースとしても使用予定

## 3種類の資材

| 種類 | URL | 用途 |
|---|---|---|
| **スライド** | `/` または `/index.html` | 保健師など専門職が、保護者に説明するときに使う（全17枚、16:9） |
| **ポスター** | `/poster.html` | 健診会場や園、自治体窓口に貼る（A3縦、9テーマ一覧） |
| **リーフレット** | `/leaflet.html` | 健診で保護者に渡す（A4両面、9テーマ＋家庭でできること） |

## スライドの使い方

- 右上「話し手モード」ボタンで各スライド下の話し手用スクリプト表示／非表示
- キーボード `S` でも話し手モードを切替可能
- 右上「印刷」ボタンでA4横印刷
- 右上「ポスター」「リーフレット」のリンクで他資材へ移動

## デザインシステム

3軸を色分けで統一しています。

- **こども（主体、権利、発達）** — オレンジ #E8742C
- **生活、健康（育ちの基盤）** — ティールブルー #3E8FAB
- **環境、社会（支える仕組み）** — セージグリーン #7FA982
