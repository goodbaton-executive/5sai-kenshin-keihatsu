# 5歳児健診でつなぐ こどもまんなか社会

日本小児保健協会 健やか親子21・成育医療等基本方針推進委員会の提言「5歳児健康診査を活用した『こどもまんなか社会』の実現に向けた啓発の推進」（2026年5月21日）に基づく、**保護者向け啓発スライド**のプレビュー。

## 位置づけ

- **委員会内プレビュー**（限定共有）
- 保健師など専門職が、保護者に説明するときに使うスライドのドラフト
- 検索エンジンには掲載されない設定（noindex、robots.txt）
- 後続成果物（ポスター、リーフレット）の内容ソースとしても使用予定

## 構成

全17スライド（表紙＋本文15＋裏表紙）。

- 5歳児健診の意義（#1〜#3）
- Take Home Message（#4、#14）
- 9つのテーマ（#5〜#13）
- 健診後の支援（#15）
- 今日からできる5アクション（#16）

## 使い方

- 右上「話し手モード」ボタンで各スライド下の話し手用スクリプト表示／非表示
- キーボード `S` でも話し手モードを切替可能
- 右上「印刷」ボタンでA4横印刷（PDFも生成可能）
